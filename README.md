Unity UI System

This project contains basic UI elements built in Unity, including menu, dialogue, and quest panels.

---

Features

- Main menu UI
- Dialogue panel
- Quest panel
- Popup messages
- Automatic NPC interaction

---

How to Run

1. Open the project in Unity Hub
2. Open the scene from:
   Assets/Scenes/
3. Press Play

---

Controls

- Walk near the NPC → Interaction triggers automatically

---

Notes

- Some large assets are not included
- Unity may automatically re-import missing packages
